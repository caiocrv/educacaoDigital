Aula Introdutória: Fundamentos de HTML

1. O que é HTML?

HTML é a sigla para HyperText Markup Language, ou seja, Linguagem de Marcação de Hipertexto. É a base da estrutura de páginas web. Com HTML, criamos títulos, parágrafos, links, imagens e muito mais. Ele organiza o conteúdo para que o navegador saiba como exibir as informações.

2. Como criar parágrafos:

Para escrever textos ou blocos de conteúdo, usamos a tag <p>, que representa um parágrafo. É uma das estruturas mais comuns em qualquer site. 

<p>Este é um parágrafo em HTML.</p>

Outras tags de conteúdo: <h1> a <h6> (títulos), <div> (blocos genéricos), <span> (texto em linha).

3. Criando links com a tag <a>:

Para criar um link clicável, usamos a tag <a> e o atributo href, que indica o destino do link. 

<a href="https://www.exemplo.com">Visite o site</a>

Outros atributos importantes: target="_blank" para abrir em nova aba, title para dica ao passar o mouse.

4. Inserindo imagens com a tag <img>:

Para exibir imagens em uma página, usamos a tag <img>. Ela precisa do atributo src para indicar o caminho da imagem, e do atributo alt para acessibilidade. 

<img src="caminho/da/imagem.jpg" alt="Descrição da imagem">

A tag <img> não tem fechamento (é uma tag vazia).

5. Comentários em HTML:

Os comentários servem para adicionar anotações no código, que não são exibidas ao usuário no navegador. Eles são úteis para organização, explicações ou desativação temporária de partes do código. 

<!-- Este é um comentário em HTML -->

Evite usar //, /* */ ou ##, pois esses formatos pertencem a outras linguagens como JavaScript, CSS e Python.
