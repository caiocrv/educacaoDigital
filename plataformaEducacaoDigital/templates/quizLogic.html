Aula Introdutória: Lógica de Programação

1. O que é um algoritmo?

Um algoritmo é uma sequência lógica e ordenada de passos que resolve um problema ou executa uma tarefa. Podemos compará-lo a uma receita de bolo: cada passo deve ser seguido corretamente para obter o resultado esperado. 

Exemplo: 
  1. Pegue dois números.
  2. Some os dois.
  3. Mostre o resultado.

2. Laços de repetição (ou loops):

São estruturas usadas para repetir blocos de código enquanto uma condição for verdadeira. Isso evita a repetição manual de comandos e torna o código mais eficiente. 

# Exemplo em Python:
  for i in range(3):
      print("Olá!")  # Saída: Olá! (3 vezes)

Outros exemplos de laço incluem: while e do...while (em algumas linguagens).

3. Estruturas condicionais:

Permitem executar diferentes blocos de código com base em condições. A estrutura mais comum é o if, que verifica se algo é verdadeiro. 

idade = 18
  if idade >= 18:
      print("Você é maior de idade.")

O if pode ser combinado com else e elif (em Python) para testar múltiplas possibilidades.

4. Variáveis:

Uma variável é um espaço na memória usado para armazenar dados que podem mudar durante a execução do programa. Ela tem um nome e pode guardar diferentes tipos de valores: números, textos, listas, etc. 

nome = "Ana"
  idade = 20

Com elas, podemos manipular e reutilizar valores com facilidade.

5. Ordem das operações:

Assim como na matemática, em programação existe uma ordem para executar operações. Primeiro multiplicações e divisões, depois somas e subtrações.

Na expressão x = 5 + 3 * 2, primeiro é feito 3 * 2 e depois somado com 5. Resultado: 11. 

x = 5 + 3 * 2
  print(x)  # Saída: 11

Usar parênteses pode alterar a ordem, como em (5 + 3) * 2 = 16.
