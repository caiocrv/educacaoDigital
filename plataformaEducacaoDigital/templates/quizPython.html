Aula Introdutória: Fundamentos Essenciais de Python

1. A função input():

Em Python, a função input() serve para capturar informações digitadas pelo usuário. Independentemente do que o usuário digitar — um número, uma palavra ou uma frase — o valor retornado será sempre do tipo str (string).

nome = input("Qual é o seu nome? ")
  print("Olá, " + nome)

Se você quiser trabalhar com números inteiros, precisará converter esse valor usando int():

idade = int(input("Digite sua idade: "))
  print(idade + 1)

2. Definindo funções com def:

Funções são blocos reutilizáveis de código. Em Python, usamos a palavra-chave def para criar uma função. Ela deve ser seguida do nome da função e parênteses (podem ou não conter parâmetros).

def saudacao():
      print("Bem-vindo(a) ao curso de Python!")

  saudacao()  # Chamada da função

3. Multiplicação de strings:

Em Python, é possível multiplicar uma string por um número inteiro. Isso significa repetir o conteúdo da string diversas vezes. Veja:

mensagem = "a" * 3
  print(mensagem)  # Saída: aaa

Esse recurso é útil para criar padrões, separadores ou repetir mensagens.

4. Dicionários em Python:

Um dicionário é uma estrutura de dados que armazena pares de chave e valor. Ele é declarado com chaves {} e permite acessar os valores por suas respectivas chaves.

aluno = {
      "nome": "Lucas",
      "idade": 21,
      "curso": "Python"
  }

  print(aluno["nome"])  # Saída: Lucas

Diferente de listas, onde os itens são acessados por índice, no dicionário usamos chaves personalizadas.

5. Operadores lógicos - and, or, not:

Python possui operadores lógicos para combinar condições:

  • and – Retorna verdadeiro se ambas as condições forem verdadeiras.
  • or – Retorna verdadeiro se pelo menos uma das condições for verdadeira.
  • not – Inverte o valor lógico (True vira False e vice-versa).
idade = 22
  tem_carteira = True

  if idade >= 18 and tem_carteira:
      print("Pode dirigir!")

Nesse exemplo, a pessoa só poderá dirigir se for maior de idade e tiver carteira de motorista.
